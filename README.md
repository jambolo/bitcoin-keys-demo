# Bitcoin Keys Demo

An interactive Bitcoin cryptography demo for exploring key generation, encoding, and address derivation.

## Features

- **Private Key Generation:** Generate and inspect Bitcoin private keys in multiple formats.
- **Public Key Derivation:** View public keys derived from private keys, with encoding options.
- **Address Creation:** Derive Bitcoin addresses (P2PKH, P2WPKH, etc.) from public keys.
- **Mini Key Demo:** Explore the "mini private key" format.
- **Seed Phrase (BIP39):** Generate and inspect BIP39 mnemonic seed phrases and their derived keys.
